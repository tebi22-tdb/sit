Formato JSON. Es un formato ligero de intercambio de datos. Leerlo y e scribirlo 
es simple para humanos, mientras que para las máquinas es simple 
interpretarlo y generarlo. Está basado en un subconjunto de JavaScript , 
Standard ECMA- 262, ideal para el intercambio de datos (JSON, 2021). 
 
Es una cadena cuyo formato recuerda al de los objetos literales 
JavaScript. Incluye tipos de datos básicos dentro como, cadenas, números, 
arreglos, booleanos, y otros literales de objeto, esto permite construir un a 
jerarquía de datos (Mozilla, 2021). 
 
a) Colección de pares. En varios lenguajes esto es conocido como 
un objeto, registro, estructura, diccionario, tabla hash, lista de claves o un 
arreglo asociativo (Mozilla, 2021). 
 
b) Lista ordenada de valores . En la mayoría de los lenguajes, esto se 
implementa como arreglos, vectores, listas o secuencias (Mozilla, 2021).

2.12.2.4 Pruebas 
 
1) Almacenamiento de archivo (Storage) . Es un servicio de almacenamiento de 
objetos al que puede acceder a través de Google Cloud Platform. Con  ello ir 
a los archivos a través de referencias, cargar archivos fácilmente y también 
monitorear el progreso de las tareas (Firebase, 2021). 
 
Ofrece un servicio de almacenamiento de objetos potente, simple y rentable  
construido para el escalamiento de Google. Los SDK de Firebase para 
Cloud Storage agregan la seguridad de Google a las operaciones de carga y 
descarga de archivos de tus apps de Firebase, sin importar la calidad de la 
red (Firebase, 2021). 
 
2) Cloud Firestore (Firestore en la nube) . Es una base de datos flexible y 
escalable para el desarrollo en servidores, dispositivos móviles y la Web 
desde Firebase y Google Cloud (Firebase, 2021).

(Firebase, 2021).

Twitter, entre otros. Se integra estrechamente con otros servicios y

(Firebase, 2021).

web completamente administrado para microservicios y contenido estático y 
dinámico. Cuenta con el respaldo del almacenamiento global (Firebase, 
2021). 
 
5) Funciones (cloud functions). Es un framework sin servidores que permite 
ejecutar de forma automática el código de backend en respuesta a las 
solicitudes HTTPS.  El código JavaScript o TypeScript se almacena en la 
nube de Google y se ejecuta en un entorno administrado (Firebase, 2021).
